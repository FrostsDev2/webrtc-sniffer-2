# webrtc-sniffer-2
Web RTC packet capture for sites like Umingle, Omegle, OME.tv, and more.
# Features 
Shows IPv4/IPv6, geo location, ISP (Internet Provider), ASN detection, VPN detection, Tor exit node id.
# UI
Floating overlay, Dark themed UI, Displays peers IP, location, ISP, VPN, Tor, Leaflet.js map showing peers ISP Location
# Data Export
Export collected data out of the panel.
